# knrhds.com

Personal portfolio site for Ken Rhodes. Civic tech and community projects
based in Saint Paul, Minnesota.

Live at: https://knrhds.com

## Tech stack

- [Astro](https://astro.build) + [Tailwind CSS](https://tailwindcss.com) + [DaisyUI](https://daisyui.com)
- [Cloudflare Pages](https://pages.cloudflare.com) for hosting
- Based on the [Astrofy](https://github.com/manuelernestog/astrofy) template by Manuel Ernesto Garcia

## Local development

```sh
pnpm install
pnpm dev
```
